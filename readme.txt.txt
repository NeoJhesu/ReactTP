# Mi Proyecto

Este es un proyecto que utiliza JS, React, HTML y CSS para crear una página web interactiva.

Cuenta con un formulario, una página de detalles de productos, carrito de compras y una página principal que te permite seleccionar entre los distintos productos. 
Al seleccionar alguno de los productos, estos te redirigen a la página de detalles en la cual te da la opción de elegir la cantidad deseada de los productos y agregarlos al carrito. Una vez hayas agregado algo al carrito, 
la misma te avisará por alerta que el producto se agregó al mismo.

Para ver los productos agregados en el carrito, vas a seleccionar en la barra de navegación el icono el cual te redirige al mismo. 
A su vez, también en la misma barra de navegación se encuentra el icono de Home el cual te envía directamente a la página de inicio. 
Ya en la página de inicio se puede utilizar la barra de búsqueda que se encuentra en el navegador para poder buscar los productos por nombre o modelo.

En el formulario vamos a encontrar con la solicitud de datos del usuario, en el cual si el usuario no escribe un nombre mayor a 4 caracteres el botón de enviar no aparece.
